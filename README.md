# thelifeofa - Blog

## Stack

- Gridsome
- GraphCMS

Blog built with Gridsome, making use of local markdown files for blog posts.

These can be categorised as either `dev` or `human`

## Requirements

- Node@10
- Yarn

## Setup

- `yarn`
- `yarn dev`
